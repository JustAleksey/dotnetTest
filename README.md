# dotnetTest

Минимальный проект на .NET 10 для экспериментов и обучения.

## Обзор

dotnetTest — небольшой проект, предназначенный для демонстрации типичных рабочих процессов разработки: сборка, запуск, отладка и тестирование с помощью .NET SDK и Visual Studio. Репозиторий преднамеренно простой, чтобы использоваться для быстрых экспериментов и обучения.

## Требования

- .NET 10 SDK (https://dotnet.microsoft.com)
- Visual Studio 2026 (или новее) или Visual Studio Code
- PowerShell (рекомендуется на Windows)

## Быстрый старт

1. Клонируйте репозиторий:

   git clone https://github.com/JustAleksey/dotnetTest.git
   cd dotnetTest

2. Восстановите пакеты и соберите проект:

   dotnet restore
   dotnet build

3. Запустите приложение:

   dotnet run --project ./

4. Запустите тесты (если есть):

   dotnet test

## Открытие в Visual Studio

Откройте решение (.sln) в Visual Studio 2026 и используйте IDE для сборки, запуска и отладки. Убедитесь, что в установке Visual Studio добавлена поддержка .NET 10.
